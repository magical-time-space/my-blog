              # 课程视频介绍

              # 课程概述

              本课程讲授色彩学、色度学基础理论；在此基础上讲述图像复制过程中色彩分解、传递、组合的规律；色彩还原质量评价及控制，影响色彩变化因素的一门课程。通过本课程的学习，学生将全面学习理解颜色、描述颜色、复制颜色等相关知识。为后续印刷专业课程的学习打下坚实的基础。

              # 授课目标

              通过颜色科学课程的学习，使学习者掌握色彩基础理论、色度学基础理论，具备应用色彩学基础理论分析印刷及颜色相关领域实际问题的能力，并具备一定的颜色鉴赏能力；同时具备应用色度学基础理论开展颜色相关领域科学研究的能力。为从事颜色设计、颜色复制、色彩管理、光谱颜色分析等等工作打下一定基础。

              # 课程大纲

              01 颜色及颜色形成 学生理解颜色，理解颜色在生活以及印刷行业中作用，颜色形成的四要素。 课时

              • 1.1 颜色定义及颜色与衣食住行
              • 1.2颜色形成的物理基础-色光三原色
              • 1.3物体显色机理
              • 1.4颜色的物体基础-色料三原色
              • 颜色及颜色形成

              02 颜色感觉的形成及应用 颜色与人的心理以及生理的关系。颜色在印刷工业中作用。 课时

              • 2.1色彩生理基础-人眼
              • 2.2颜色视觉障碍
              • 2.3彩色油墨调配
              • 颜色感觉的形成及应用

              03 颜色视觉理论及颜色现象 人眼观察颜色特点，在印刷工业中如何利用视觉特点进行色彩设计。学习颜色视觉理论等 课时 颜色视觉理论

              • 3.2颜色视觉现象
              • 颜色视觉理论及颜色现象

              04 颜色的显色系统 从色度学角度表征颜色，实现颜色的量化表征，为学生描述颜色奠定基础。 课时

              • 4.1 颜色的心理三属性
              • 4.2 孟塞尔颜色系统
              • 4.3 自然颜色系统
              • 第四周单元测验

              05 颜色的混色系统 学习色度学颜色系统，通过色度学相关公式完成颜色计算等。 课时

              • 5.1 颜色匹配与三刺激值
              • 5.2 CIE1931RGB表色系统
              • 5.3 CIE1931XYZ表色系统
              • 第五周单元测验

              06 均匀颜色空间与同色异谱 学习均匀颜色空间中颜色表征，理解同色异谱现象，为颜色科学研究奠定基础。 课时

              • 6.1 CIE1976Lab均匀颜色空间
              • 6.2 CIE1976Luv均匀颜色空间
              • 6.3 同色异谱
              • 第六周单元测验

              07 复制颜色——颜色的分解 在学习色彩理论基础上完成颜色分解，实现在印刷中应用。 课时

              • 7.1 颜色的分解概述
              • 7.2 颜色的分解方法-照相分色
              • 7.3颜色的分解方法-电分分色、DTP及CTP分色
              • 7.4颜色分色过程1
              • 7.5颜色分色过程2
              • 颜色分解

              08 复制颜色——颜色的合成 学习网点基础知识，掌握颜色合成的基本规律及原理，为揭示成色奠定基础。 课时

              • 8.1 网点定义及作用.
              • 8.2 网点面积率
              • 8.3 网点扩大
              • 8.4颜色合成
              • 网点及颜色组成
